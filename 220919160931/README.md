## Moving on the page
* `set relativenumber`: blazing fast to reach a line without any pattern
* `<C-d> <C-u>`: scroll half of the window 
* `[m`, `[M`, `]m`, `]M`: to jump to the beginnin { or end } of a function

## Moving on the line
* `web` (no comment)
* `f/F + <char>`: jump to char
* `$`: go to the end
* `^`: go to the begining
